ZipCode Resolver
http://webservices.eraserver.net/zipcoderesolver/zipcoderesolver.asmx?WSDL
Provides zip code information for a US address  This web service allows you to validate partial United States mailing addresses against the U.S. Postal Service database. You can enter street, city, and state information and get various return values including: five-digit standard ZIP code (NNNNN), ZIP+4 code (NNNNN-NNNN), and Corrected Address (complete street, city, state, and ZIP+4). You can retrieve the corrected address in HTML form (for easy display on a web page) or in an XML structure that includes Street, City, State, ShortZip, and FullZip elements.
