ISOcode2shortformat
http://www.siliconllama.com/services/wsdl/ISOcode2shortformatService.wsdl
Given a ISO 3166 country code, this will return the format for that   locale's short date.  IE: if the country code is US, it will return: mm/dd/yyyy .   Likewise, if the country code is UK, it will return: dd/mm/yyyy .
