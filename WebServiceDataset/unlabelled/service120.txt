RSA Functions
http://ntg.webs.innerhost.com/RSAFunctions/RSAFuncs.asmx?WSDL
RSA Functions is a web service that implements some generic methods   which utilize the RSA cryptography provider of the .NET framework.   The web service can issue private/public RSA keys and perform   encryption and decryption operations of hexadecimal data using the   keys.
