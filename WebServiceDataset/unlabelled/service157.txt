ISOcodes
http://www.siliconllama.com/services/wsdl/ISOcodesService.wsdl
This service is intended to resolve ISO 3166 2 letter country   codes with their English names.    The output is an array of arrays. The top level array is a list of   all the values returned. The inside array is made up of the country   code and its fullname.
