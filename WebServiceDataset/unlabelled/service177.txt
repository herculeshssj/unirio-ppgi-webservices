AmazonBox
http://www.xmlme.com/WSAmazonBox.asmx?WSDL
AmazonBox uses Amazon.com Web Services to return a string formatted   HTML table with up to 9 product links and their associated prices,   images, and manufacturers. All boxes need only a single string input   value. An empty HTML table is returned if no results can be   retrieved.
