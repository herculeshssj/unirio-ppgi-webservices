Html2Xml
http://www.dev1.eraserver.net/REFLECTIONIT/Html2xml.asmx?WSDL
The Html2Xml webservice takes an Html text and converts it into an   Xml text. Some corrections are done to the Html to make it well-  formed Xml.     You can use this webservice to screen scrape a web page and convert   it to Xml. The Xml can then be used for further processing.
