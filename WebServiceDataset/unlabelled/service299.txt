XMethods Query Service
http://www.xmethods.net/wsdl/query.wsdl
This service provides a SOAP RPC query interface to XMethods.  It   allows you to access both summary and detail information about the   services, and allows you to filter services by publisher.      This service supercedes the old &quot;XMethods Listings Service&quot;.
