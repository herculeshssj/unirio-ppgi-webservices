Live World Cup Scores
http://scores.serviceobjects.com/CupScores.asmx?WSDL
DOTS for World Cup Live is a programmable XML service that delivers   the real-time scores and game times of all matches of the 2002 World   Cup.  This web service will allow you track the ongoing scores of all   the World Cup games.    DOTS for World Cup Live is designed for web sites, this web service   with optional Excel-based client provides real time score on your   site. DOTS for World Cup Live Web Service allows for flexible   integration for high volume web sites.
