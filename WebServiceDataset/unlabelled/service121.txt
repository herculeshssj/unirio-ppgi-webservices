SpamKiller
http://wavendon.dsdata.co.uk/axis/services/SpamKiller?wsdl
Multi-purpose web service for manipulating POP3/IMAP mailboxes.  Intended purpose for removing spam.      You specify spam criteria (email addresses, subjects) and it removes   emails that match.  (Subjects are specified as regular expressions).    Has a test mode which should be used first which reports what would   happen if you ran it for real!
