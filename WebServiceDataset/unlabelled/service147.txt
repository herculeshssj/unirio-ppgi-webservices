GetCurrencyExchange
http://www.atlaz.net/webservices/GetCurrencyExchange.wsdl
Takes 3 inputs:  number =&gt; number of units to be exchanged (obviously set to 1 for the   current rate)  currency1 =&gt; ISO code for the currency to be exchanged FROM  currency2 =&gt; ISO code for the currency to be exchanged TO    Returns a number. If the ISO codes are not present, an error will   occur.
