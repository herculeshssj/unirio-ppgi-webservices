SearchMusicTeachers
http://www.PerfectXML.net/WebServices/MusicTeachers/MusicTeachers.asmx?wsdl
Locates music teachers by zip code.  This Web Service can be used to locate music teachers based on Zip code and various other optional conditions, like Instrument, Style, Skill Level, and Radius distance from the specified Zip. This Web Service is created by PerfectXML Team.
