StockQuotes Web Service
http://www.gama-system.com/webservices/stockquotes.asmx?wsdl
Web Service enables programmatic access to stock values on different   stock exchanges. Currently we support Ljubljanska Stock Exchange   (http://www.ljse.si) and NASDAQ (http://www.nasdaq.com). The service   offers past values of stocks, and returns it in XML. Multiple   querying functions supported (date based, value based, indexes,   single value return, ...)    The service is registered on the Microsoft UDDI repository.
