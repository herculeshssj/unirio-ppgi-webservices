Alan Bush Compositions
http://www.alanbushtrust.org.uk/soap/compositions.wsdl
Interface to the Alan Bush Compositions Database.    The Alan Bush Compositions Web Service provides a SOAP   interface to the Compositions Database of the British   composer Alan Bush who lived from 1900 to 1995. The   database in HTML format can be viewed at:  http://www.alanbushtrust.org.uk    It provides 2 methods:  GetCategories - Provides a list of composition categories.  GetCompositions - Provides a list of compositions for a particular   category.
