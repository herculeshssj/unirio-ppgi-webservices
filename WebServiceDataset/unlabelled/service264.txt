U.S. CFR Section Text  (CFRSect)
http://www.oakleaf.ws/cfrsectws/cfrsectws.asmx?wsdl
One of three XML Web services that provide online Access to the   OakLeaf U.S. Code of Federal Regulations delivered as XML or XHTML   documents from a 1-GB database. (Not an official version of the U.S.   CFR).    This service returns the text of one of the 180,000+ sections of the   CFR as an XML document or a formatted XHTML page. (The printed   version of the CFR has 204 volumes.)    Go to http://www.oakleaf.ws/cfr/about.aspx for detailed descriptions   of the three XML Web services.
