MXChecker
http://beta2.eraserver.net/webservices/mxchecker/mxchecker.asmx?WSDL
Checks for valid DNS MX records. This is a simple web service that checks for valid MX records using DNS services. You can use this service to validate any email address. This service uses DNS services to query the mail server named in your email address to see if the email address is valid.
