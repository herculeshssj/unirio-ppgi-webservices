USFootballNews
http://sal006.salnetwork.com:83/Gwen/USFootballNews/CUSFootballNews.xml
This web service has two methods: GetAmericanFootballNews and AmericanFootballRecordset.<br>GetAmericanFootballNews will return an encoded XML string of news articles about American Football with links to find out more.<br>The AmericanFootballRecordset method returns the same data by using a ByRef ADO recordset instead of an XML string.
